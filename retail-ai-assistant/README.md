# Retail AI Assistant

## Running the Project

To run this project step by step on Windows (PowerShell), follow these instructions:

1. **Navigate to the project directory:**
   ```powershell
   cd "C:\Users\DELL\Desktop\New folder\retail-ai-assistant"
   ```

2. **Activate the virtual environment:**
   ```powershell
   .\venv\Scripts\Activate.ps1
   ```

3. **Install the dependencies (if you haven't already):**
   ```powershell
   pip install -r requirements.txt
   ```

4. **Run the AI assistant:**
   ```powershell
   python -m app.main
   ```

After running the final command, the interactive chat session will start in your terminal and you will see a `You:` prompt waiting for your input. To stop the assistant, simply type `exit` and hit enter.
